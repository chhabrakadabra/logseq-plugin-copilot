# Contributing

## Development

Run `npm install` to install dependencies.

Run `npm run dev` to start the plugin.

Follow [these instructions](https://gist.github.com/xyhp915/bb9f67f5b430ac0da2629d586a3e4d69) to
install this plugin in dev mode.

## Release a new version

Tag the commit on the main branch that you want to release and push the tag. This should trigger a
Github action that will create a release. Head over to the release, edit the description and make
sure to uncheck the pre-release checkbox to actually release to Logseq.
